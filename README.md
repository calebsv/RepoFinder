# 📖 Sobre o Projeto

O **RepoFinder** é uma aplicação web que consome a **API pública do GitHub** para localizar usuários e exibir seus repositórios de maneira rápida e intuitiva.

Além de apresentar informações relevantes do perfil, a aplicação organiza os repositórios em uma interface moderna com efeitos visuais, animações e design responsivo.

O projeto foi desenvolvido como forma de praticar consumo de APIs REST, manipulação do DOM, JavaScript moderno e construção de interfaces utilizando Tailwind CSS.

---

# ✨ Demonstração

### O usuário pode:

- Pesquisar qualquer usuário do GitHub
- Visualizar o perfil encontrado
- Listar todos os repositórios públicos
- Acessar diretamente qualquer repositório

- Navegar por uma interface moderna e responsiva

# 💻 Como utilizar

1. Digite o nome de um usuário do GitHub.
2. Clique em **Buscar**.
3. Aguarde o carregamento.
4. Visualize o perfil encontrado.
5. Navegue pelos repositórios.
6. Clique em **Abrir** para acessar diretamente o projeto no GitHub.
